CI CD
continuous integration continous deploiement

Intégration et déploiement continus

Nouvelle manière de travailler dans laquelle on va impliquer les équipes de développeurs et les équipes d'exploitations (ingénieurs en charge des infrastructures : hardware)

Pour faciliter le travail de ces deux équipes => CI CD

Ensemble de traitements que l'on va mettre en place qui va permettre de clarifier le transfert du code d'une équipe à l'autre

=> DevOps connaître toute la chaîne de production au déploiement
=> idée => code => test => créer des machines stockées => diffuser => client utilisation

=> durée entre l'idée et la livraison client : Time To Market

=> intégration Hell => Infernal => réduire l'intégration Hell
